Hướng dẫn Triển khai Trang HTML Tĩnh lên Vercel

Chào mừng bạn! Hướng dẫn này sẽ giúp bạn đưa trang web HTML tĩnh của mình lên mạng một cách dễ dàng bằng Vercel, sử dụng GitHub để quản lý tệp.

1

Chuẩn bị Tệp Trang Web của Bạn

Hãy đảm bảo rằng tất cả các tệp HTML, CSS, JavaScript và hình ảnh của bạn được đặt gọn gàng trong một thư mục duy nhất trên máy tính của bạn. Tệp HTML chính của bạn nên được đặt tên là index.html.

Cấu trúc thư mục ví dụ:

my-website/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── images/
    └── logo.png
[Cấu trúc thư mục ví dụ trên máy tính của bạn]
2

Tải lên Tệp của Bạn lên GitHub

Vercel hoạt động tốt nhất khi được kết nối với một kho lưu trữ Git (như GitHub). Chúng ta sẽ tạo một kho lưu trữ mới và tải tệp lên trực tiếp qua giao diện web của GitHub.

2.1. Tạo Kho lưu trữ (Repository) mới trên GitHub

  1. Truy cập GitHub.com và đăng nhập vào tài khoản của bạn (hoặc đăng ký nếu chưa có).
    2. [Trang đăng nhập/đăng ký GitHub]
  2. Trên trang chủ GitHub của bạn, tìm nút "New" (hoặc biểu tượng + ở góc trên bên phải, sau đó chọn "New repository").
    3. [Nút "New" hoặc biểu tượng "+" trên GitHub]
  3. Điền thông tin cho kho lưu trữ của bạn:
    • Repository name: Đặt một tên có ý nghĩa (ví dụ: my-static-website, personal-portfolio).
    • Description (tùy chọn): Mô tả ngắn gọn về dự án của bạn.
    • Chọn Public hoặc Private tùy theo ý muốn của bạn.
    • Tích chọn "Add a README file" (được khuyến nghị).
    4. [Trang tạo kho lưu trữ mới trên GitHub với các trường thông tin]
  4. Nhấn nút "Create repository".

2.2. Tải tệp lên Kho lưu trữ của bạn

  1. Sau khi tạo kho lưu trữ, bạn sẽ được chuyển đến trang kho lưu trữ của mình.
  2. Tìm nút "Add file" và chọn "Upload files" từ menu thả xuống.
    3. [Nút "Add file" và tùy chọn "Upload files" trên trang kho lưu trữ GitHub]
  3. Kéo và thả toàn bộ thư mục trang web của bạn (hoặc chọn từng tệp) vào khu vực tải lên. Đảm bảo bạn tải lên tất cả các tệp và thư mục con (như `css`, `js`, `images`).
    4. [Khu vực kéo và thả tệp trên GitHub]
  4. Sau khi tất cả các tệp đã được tải lên, cuộn xuống dưới cùng. Trong phần "Commit changes":
    • Thêm một tin nhắn cam kết (ví dụ: "Initial website upload").
    • Nhấn nút "Commit changes".
    5. [Phần "Commit changes" trên GitHub]
  5. Bây giờ, tất cả các tệp trang web của bạn đã nằm trên GitHub!
3

Triển khai Trang web của Bạn lên Vercel

Bây giờ, chúng ta sẽ kết nối Vercel với kho lưu trữ GitHub của bạn và triển khai trang web.

  1. Truy cập Vercel.com.
    2. [Trang chủ Vercel]
  2. Đăng nhập vào tài khoản Vercel của bạn. Nếu bạn chưa có, hãy nhấp vào "Sign Up" và chọn đăng ký bằng tài khoản GitHub của bạn. Điều này sẽ giúp quá trình kết nối dễ dàng hơn rất nhiều.
    3. [Trang đăng nhập/đăng ký Vercel với tùy chọn GitHub]
  3. Sau khi đăng nhập, trên Bảng điều khiển Vercel của bạn, tìm và nhấp vào nút "Add New..." (hoặc "New Project" ở góc trên bên phải).
    4. [Nút "Add New..." hoặc "New Project" trên Bảng điều khiển Vercel]
  4. Trong phần "Import Git Repository", Vercel sẽ hiển thị danh sách các kho lưu trữ từ nhà cung cấp Git mà bạn đã kết nối.
  5. Tìm kho lưu trữ GitHub chứa các tệp HTML tĩnh của bạn (tên bạn đã đặt ở Bước 2.1) và nhấp vào nút "Import" bên cạnh nó.
    • Nếu bạn không thấy kho lưu trữ của mình, bạn có thể cần nhấp vào "Configure GitHub App" để cấp quyền truy cập cho Vercel vào kho lưu trữ đó (hoặc tất cả các kho lưu trữ của bạn).
    6. [Danh sách kho lưu trữ trên Vercel và nút "Import"]
  6. Trên trang "Configure Project":
    • Project Name: Vercel sẽ đề xuất một tên dự án dựa trên tên kho lưu trữ của bạn. Bạn có thể thay đổi nếu muốn.
    • Root Directory: Nếu tệp index.html của bạn (và các tệp tĩnh khác) nằm ở thư mục gốc của kho lưu trữ, hãy để mặc định là ./. Nếu chúng nằm trong một thư mục con (ví dụ: public, dist), hãy chỉ định tên thư mục đó ở đây.
    • Build & Output Settings: Đối với một trang HTML tĩnh đơn giản, Vercel thường sẽ tự động phát hiện mọi thứ chính xác. Bạn thường không cần thay đổi bất kỳ cài đặt nào ở đây. "Framework Preset" có thể là "Other" hoặc "Static".
    7. [Trang "Configure Project" trên Vercel với các cài đặt]
  7. Nhấn nút "Deploy".
4

Trang web của Bạn đã Hoạt động!

Vercel sẽ bắt đầu xây dựng và triển khai dự án của bạn. Quá trình này thường chỉ mất vài giây đối với HTML tĩnh.

[Quá trình triển khai trên Vercel]

Khi hoàn tất, bạn sẽ thấy thông báo "Congratulations!" và nút "Visit".

[Thông báo "Congratulations!" và nút "Visit" sau khi triển khai thành công]

Nhấp vào nút "Visit" để xem trang web HTML tĩnh của bạn đang hoạt động trên một tên miền .vercel.app (ví dụ: your-project-name.vercel.app).

[Trang web đã được triển khai trên tên miền Vercel]

Vercel cũng cung cấp một URL xem trước duy nhất cho mỗi lần triển khai mới, điều này rất hữu ích để chia sẻ các thay đổi trước khi đưa chúng vào sản phẩm chính thức.

Bây giờ, bất cứ khi nào bạn đẩy các thay đổi mới lên kho lưu trữ GitHub đã kết nối của mình, Vercel sẽ tự động triển khai lại trang web của bạn!

Chúc mừng! Bạn đã triển khai thành công trang web HTML tĩnh của mình lên Vercel.
